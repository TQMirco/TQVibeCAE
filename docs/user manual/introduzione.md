# Introduzione

## Cos'è TQVibeCAE

TQVibeCAE è un software open source per la progettazione di **impianti elettrici**: quadri, schemi unifilari e multifilari, simbologia conforme a norme IEC/CEI.

Non è un CAD per circuiti elettronici o PCB.

## A chi è rivolto

- Progettisti impianti elettrici
- Tecnici che redigono schemi di quadro
- Chi documenta collegamenti, protezioni e conduttori
